Hash Comparisons

You can pick a correct SHA-256 value to hash1 by clicking a proper filename:

  • file1.txt
  • file2.txt
  • file3.txt

Paste the hash that you calculated to hash2





You can calculate sha-256 hash for a file in windows by using powershell and giving the command: get-filehash filename.
To select the calculated hash just double-click it and then copy/paste it to hash2 textbox.

Note: Powershell is started from cmd prompt by typing powershell or just like any other program from start-menu
